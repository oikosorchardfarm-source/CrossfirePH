# Carousel Images

Place the images you want to use in the carousel here.

## Recommended filenames (used by the current template)
- `carousel-1.svg`
- `carousel-2.svg`
- `carousel-3.svg`

You can replace these with your own `.jpg`, `.png`, or `.svg` files and update the `src` values in `index.html` accordingly.

## Weapon prize images
The spin wheel result uses these placeholders for weapon prizes:
- `weapon-1.png`
- `weapon-2.png`
- `weapon-3.png`
- `weapon-4.png`

Replace those files with the weapon images you attached (keep the same filenames) to show them in the spin result alert.
